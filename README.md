# mere

Tarjeta virtual del Día de las Madres con React (Vite). Diseño mobile-first.

## Desarrollo

```bash
npm install
npm run dev
```

## Despliegue en Vercel

Conecta el repositorio de GitHub en Vercel y usa el preset de Vite, o:

- **Build command:** `npm run build`
- **Output directory:** `dist`
